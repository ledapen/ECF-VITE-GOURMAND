# État final du projet

Cette version livre une application PHP MVC exploitable en local avec MySQL/MariaDB et MongoDB.

## Livré
- Application web MVC.
- Authentification.
- Mot de passe oublié.
- Menus + filtres AJAX.
- Commandes + calculs métier.
- Espaces utilisateur/employé/admin.
- CRUD menus, plats, horaires.
- Gestion avis.
- Gestion contact.
- Statistiques et graphique.
- Script SQL.
- Exemple MongoDB.
- Docker Compose.
- Scripts de lancement.
- Plan de tests.
- Recette fonctionnelle.
